# Vulnerability Assessment and Penetration Testing (VAPT)

## Overview
This repository documents a hands-on Vulnerability Assessment and Penetration Testing (VAPT)
exercise performed using open-source tools in a controlled lab environment.

## Objectives
- Understand security assessment without paid tools
- Perform vulnerability scanning and basic penetration testing
- Apply risk assessment using CVSS and risk matrix
- Document findings and remediation steps

## Tools Used
- Kali Linux
- OpenVAS (Greenbone)
- Nikto
- Nmap
- Metasploitable 3
- VMware Workstation

## Lab Environment
- Attacker Machine: Kali Linux
- Target Machine: Metasploitable 3
- Virtualization: VMware Workstation
