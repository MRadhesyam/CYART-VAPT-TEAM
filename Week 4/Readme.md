**Advanced VAPT**

**Overview**
This repository documents advanced Vulnerability Assessment and Penetration Testing (VAPT)
activities including exploit chaining, API security testing, privilege escalation,
network protocol attacks, mobile application testing, and a full capstone engagement.

All testing was conducted in controlled lab environments (TryHackMe, PortSwigger, VulnHub).

**Key Domains Covered**
- Advanced Exploitation
- API Security (OWASP API Top 10)
- Privilege Escalation & Persistence
- Network Protocol Attacks
- Mobile Application Security
- Full PTES-aligned Capstone Project

**Tools Used**
Metasploit, Nmap, Nikto, Hydra, Burp Suite, Postman, Responder, Ettercap,
Wireshark, MobSF, LinPEAS
